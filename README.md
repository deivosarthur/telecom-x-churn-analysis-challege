# 📊 Telecom X — Análisis de Evasión de Clientes (Churn)

Este proyecto me permitió profundizar en análisis de churn y comprender cómo variables contractuales impactan en la retención de clientes.

## 📌 Descripción del Proyecto

Telecom X presenta una tasa relevante de cancelación de clientes (Churn), lo que impacta directamente en ingresos recurrentes y estabilidad del negocio.

El objetivo de este proyecto fue realizar un análisis exploratorio de datos (EDA) para identificar patrones y variables asociadas a la evasión, sentando bases para futuros modelos predictivos y estrategias de retención.

---

## 🎯 Enfoque del Análisis

- Extracción de datos desde API en formato JSON.
- Normalización de estructuras anidadas.
- Proceso ETL (limpieza, transformación y validación).
- Análisis exploratorio orientado a negocio.
- Identificación de variables críticas para churn.

## 🔎 Principales Hallazgos

### 📉 Tasa General de Churn:
La evasión alcanza aproximadamente un 26.6%, lo que implica que 1 de cada 4 clientes cancela el servicio.

### 📄 Tipo de Contrato:
El contrato es la variable más determinante:

- Month-to-month → 42.7% churn
- One year → 11.3%
- Two year → 2.8%

Contratos de mayor duración reducen significativamente el riesgo.

### ⏳ Antigüedad (Tenure):
Clientes con menor antigüedad presentan mayor probabilidad de cancelar.
Los primeros meses del ciclo de vida del cliente son críticos.

### 💰 Cargos Mensuales
Clientes con cargos mensuales más elevados muestran mayor tendencia a evasión.

---

## 💡 Insights Estratégicos

El perfil de mayor riesgo corresponde a:

- Contrato mensual
- Baja antigüedad
- Cargos mensuales altos

Esto sugiere que las estrategias de retención deben enfocarse principalmente en clientes nuevos con contrato flexible.
